Properties of Flex container

  • Display flex / Inline-flex
  • Flex-direction
  • Flex-wrap
  • Flex-flow
  • justify-content
  • align-items

    • Display flex / Inline-flex

  • Flex is the value of css display . By using display flex in parent element, child elements automatically align like Column or Row

    • Example -:
    box
    box1
    box2
    box3

    Flex Direction

  • Flex Direction property is given to flex container to change direction of flex items. By default, flex direction is row.

    • Flex Directions value

    1. Row ( default)
    2. Row-reverse
    3. Column
    4. Column-reverse

    Example -:

    1. Flex Direction - Row

      2. box
      box1
      box2

    2. Flex Direction - Row-Reverse

      3. box
      box1
      box2

    3. Flex Direction - Column

      4. box
      box1
      box2

    4. Flex Direction - Column-Reverse

      5. box
      box1
      box2

    Flex Wrap

  • Flex items will always fit in one row even if content is more. flex wrap is used to allow items to wrap in next line or wrap in reverse direction.
